## Tooltip

You can show a tooltip by adding the `tooltip` prop. This property
defaults to false since most places in the application do not need it.

## Label

You can show the avatar's name as a label next to the icon by passing the
`labelled` prop. You can also show subtitle text under the label by passing the
`subtitle` prop.

<Canvas>
  <Story name="Label">
    <Row>
      <Avatar labelled size="s" name="Grumpy&nbsp;Cat" image={fake.profilePicture()} style={moreSpace} />
      <Avatar labelled size="m" name="Grumpy Cat" image={fake.profilePicture()} style={moreSpace} />
      <Avatar labelled size="l" name="Grumpy Cat" image={fake.profilePicture()} style={moreSpace} />
      <Avatar labelled size="xl" name="Grumpy Cat" image={fake.profilePicture()} style={moreSpace} />
    </Row>
    <Row>
      <Avatar labelled subtitle="Hero of cats" name="Grumpy Cat" size="s"  style={moreSpace} />

      <Avatar labelled subtitle="Hero of cats" name="Grumpy Cat" size="l"  style={moreSpace} />
      <Avatar labelled subtitle="Hero of cats" name="Grumpy Cat" size="xl" style={moreSpace} />
    </Row>
  </Story>
</Canvas>

If you want to show some different text in the label area, you can instead pass
the `title` prop.
