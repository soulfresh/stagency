### Tight

You can tighten up the padding inside of the FieldSet by passing the `tight`
prop. This can be useful for content that doesn't need space for error messages
such as AvatarSearch components or radio/checkbox groups.

<Canvas>
  <Story name="Tight">
    <Fieldset
      legend="Tight Fieldset"
      tight
    >
      <Input tight placeholder="Enter Yes if you like dogs"/>
      <Input tight placeholder="Enter Yes if you like cats"/>
    </Fieldset>
  </Story>
</Canvas>

### Removable

If the FieldSet is part of a list, you can pass the `onDelete` callback prop
to add a delete button to the FieldSet.

<Canvas>
  <Story name="Delete">
    <Fieldset
      legend="Delete Fieldset"
      tight
      onDelete={() => action('onDelete')}
      deleteText="Delete Pet Chooser"
    >
      <Input tight placeholder="Enter Yes if you like dogs"/>
      <Input tight placeholder="Enter Yes if you like cats"/>
    </Fieldset>
  </Story>
</Canvas>
