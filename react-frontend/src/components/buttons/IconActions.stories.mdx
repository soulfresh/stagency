# Icon Action

The following icon buttons are available in this project.
They are all `<Action>` components, so you can pass any Action
prop to these icon buttons.

## With Children

Any children you pass will be placed next to the icon.
You can switch the orientation of the icon and children using
the direction prop.

<Canvas>
  <Story name="With Children">
    <div style={{ display: "flex", alignItems: "center" }}>
      <CloseAction>Direction Right</CloseAction>
      <span> | </span>
      <CloseAction direction="left">Direction Left</CloseAction>
    </div>
  </Story>
</Canvas>

"s", "m" or "l".
